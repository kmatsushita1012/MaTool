# 行動表出力追加　仕様書

## 概要

MaToolが管理する祭では屋台が通過する町をリストアップしPDFで出力する必要がある
現在までに`RoutePassage`を入力してDBに保存できるようにした
今回はこの`RoutePassage`をPDFに出力できるようにする

## 出力仕様
- 表形式で出力する
 - 画面仕様に合わせてA4縦で出力する
 - 表のレイアウトは既存の実装に合わせて調整する
- 各日付ごとにページを分けて出力する

| 項目名 | 位置 | 備考 |
| --- | --- | --- |
| 日付 | 左上 | DD日 |
| 町名 | 右上 | `"町名（${District.name}区）"` |
| 時間帯 | 表左側 | `Period.title` 縦書き |
| 通過町 | 表 | `RoutePassage.memo` or `RoutePassage.districtId`より`District.name` |

##　出面
主に既存の行動図出力と合わせて出力する

- `HeadquarterDistrictDetailView`
    - 経路図一括出力
        - 既存のPDF出力に追加
        - 行動表 -> 行動図の順
        - 提出資料出力 に命名変更
    - 行動表出力
        - 新規
            - 経路図一括出力の下に増設
        - 行動表のみ出力する
- `HeadquarterDistrictListView`
    - 経路図一括出力
        - 既存のPDF出力に追加
        - 各町ごと行動表 -> 行動図の順
        - 提出資料出力 に命名変更
        - 各町ごとに別のファイルで出力する
- `DistrictDashboardView`
    - 提出資料出力
        - 新規
            - ルートセクションの下に新規セクションで追加
        - 行動表 -> 行動図の順
        - 提出資料出力 に命名変更
    - 行動票出力
        - 新規
            - 提出資料出力と同じセクション
        - 行動表のみ出力する
